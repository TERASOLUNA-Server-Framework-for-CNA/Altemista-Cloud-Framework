To manually install the {framework} Config Server feature, add the following dependencies to the pom.xml file of the aggregator boot project of your application:

[source,xml]
.new-application-boot/pom.xml
----
<dependency>
	<groupId>org.altemista.cloudfwk.microservices</groupId>
	<artifactId>altemista-cloudfwk-core-microservices-config-conf</artifactId><!--1-->
</dependency>
----
<1> Config Server feature dependencies and auto-discoverable configuration.

The {framework} `AltemistaCloudfwkMicroserviceConfig` class has the next annotations to configure the application as a {framework} Config Server

[source,java]
----
include::{basedir}/altemista-cloudfwk-core-microservices-config-conf/src/main/java/org/altemista/cloudfwk/config/microservice/config/AltemistaCloudfwkMicroserviceConfig.java[tags=annotation]
----
<1> The Spring Cloud @EnableConfigServer annotation is used to enable the application as a Config Server.
