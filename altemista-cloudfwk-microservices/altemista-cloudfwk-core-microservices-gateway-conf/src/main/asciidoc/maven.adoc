To manually install the {framework} Gateway Server feature, add the following dependencies to the pom.xml file of the aggregator boot project of your application:

[source,xml]
.new-application-boot/pom.xml
----
<dependency>
	<groupId>org.altemista.cloudfwk.microservices</groupId>
	<artifactId>altemista-cloudfwk-core-microservices-gateway-conf</artifactId><!--1-->
</dependency>
----
<1> Gateway Server feature dependencies and auto-discoverable configuration.

The {framework} `AltemistaCloudfwkMicroserviceGateway` class has the next annotations to configure the application as a {framework} Gateway Server

[source,java]
----
include::{basedir}/altemista-cloudfwk-core-microservices-gateway-conf/src/main/java/org/altemista/cloudfwk/config/microservice/gateway/AltemistaCloudfwkMicroserviceGateway.java[tags=annotation]
----
<1> The Spring Cloud Netflix @EnableZuulProxy annotation is used to enable the application as a Gateway Server.
