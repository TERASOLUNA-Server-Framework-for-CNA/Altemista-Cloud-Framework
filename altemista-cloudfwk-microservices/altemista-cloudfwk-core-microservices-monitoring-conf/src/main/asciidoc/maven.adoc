To manually install the {framework} Monitoring Server feature, add the following dependencies to the pom.xml file of the aggregator boot project of your application:

[source,xml]
.new-application-boot/pom.xml
----
<dependency>
	<groupId>org.altemista.cloudfwk.microservices</groupId>
	<artifactId>altemista-cloudfwk-core-microservices-monitoring-conf</artifactId><!--1-->
</dependency>
----
<1> Monitoring Server feature dependencies and auto-discoverable configuration.

The {framework} `AltemistaPlusMicroserviceMonitoring` class has the next annotations to configure the application as a {framework} Monitoring Server

[source,java]
----
include::{basedir}/altemista-cloudfwk-core-microservices-monitoring-conf/src/main/java/org/altemista/cloudfwk/config/microservice/monitoring/AltemistaCloudfwkMicroserviceMonitoring.java[tags=annotation]
----
<1> The `@EnableHystrixDashboard` annotation is used to enable Hystrix Dasboard 
<2> The `@EnableCloudDashboard` annotation is used to enable Cloud Dashboard
<3> The `@EnableTurbine` annotation is used to enable Turbine as a Server
