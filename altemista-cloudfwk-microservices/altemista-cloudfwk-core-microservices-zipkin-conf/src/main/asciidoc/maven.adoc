To manually install the {framework} Zipkin Server feature, add the following dependencies to the pom.xml file of the aggregator boot project of your application:

[source,xml]
.new-application-boot/pom.xml
----
<dependency>
	<groupId>org.altemista.cloudfwk.microservices</groupId>
	<artifactId>altemista-cloudfwk-core-microservices-zipkin-conf</artifactId><!--1-->
</dependency>
----
<1> Zipkin Server feature dependencies and auto-discoverable configuration.

The {framework} `AltemistaCloudfwkMicroserviceZipkin` class has the next annotations to configure the application as a {framework} Zipkin Server

[source,java]
----
include::{basedir}/altemista-cloudfwk-core-microservices-zipkin-conf/src/main/java/org/altemista/cloudfwk/config/microservice/zipkin/AltemistaCloudfwkMicroserviceZipkin.java[tags=annotation]
----
