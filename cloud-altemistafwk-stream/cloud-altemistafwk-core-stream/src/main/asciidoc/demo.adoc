*ALTEMISTA Cloud Framework* ({framework}) provides the features to build a Spring Cloud Stream application.

On this tutorial we will make a simple application with Spring Cloud Stream developed with {framework} version from *{project-version}*.
Spring Cloud Stream is a framework for building highly scalable event-driven microservices connected with shared messaging systems.

More information about Spring Cloud Stream https://docs.spring.io/spring-cloud-stream/docs/current/reference/htmlsingle/[here^].
