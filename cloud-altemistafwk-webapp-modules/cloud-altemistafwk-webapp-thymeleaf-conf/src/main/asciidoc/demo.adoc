*ALTEMISTA Cloud Framework* ({framework}) provides the features to build a Thymeleaf application in a boot and web application project.

On this tutorial we will make some applications with Thymeleaf developed with {framework} version from *{project-version}*.
Thymeleaf is a Java-based library used to create a web application. It provides a good support for serving a XHTML/HTML5 in web applications

More information about Thymeleaf click on https://www.thymeleaf.org/[here^].
